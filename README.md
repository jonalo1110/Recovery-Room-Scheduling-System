## Features

- Staff can book **same-day or future** recovery room time slots  
- Each booking includes **member name**, **date**, and **start time**  
- **Today view** shows all bookings for the current day  
- **Upcoming view** displays future bookings (e.g., next 7 days)  
- Enforces a **maximum of 6 people per time slot** at the backend level  
- Logs **amenities used** (all amenities or selected ones like cold plunge, sauna, and Normatech)  
- Designed for use on a **front-desk tablet or mobile device**  
- Built with a scalable backend ready for **notifications and analytics**

## Tech Stack

- **Frontend:** Flutter & Dart  
- **Backend:** Node.js & Express  
- **Database:** PostgreSQL
